# AddHealthGroup

## Краткое описание
Добавляет группу здоровья студента.

## Входные параметры
Название | Тип данных | Описание
:---------:|:----------:|:--------:
StudentGuid|string      |Идентификатор студента
TeacherGuid|string      |Идентификатор преподавателя
HealthGroup|HealthGroupType|Группа здоровья
